Target Audience

The Target Audience for this Weather Website will be persons that want a direct recommendation for daily decisions like A need a coat, umbrella or something similar? An ordinary person that lived in one of the cities that the site covered: Preston, Soda Springs, and Fish

Personas

This site will be optimized for two types of people. First, the mother needs the information to make decisions and for ordinary citizens looking for simple weather information.

Scenarios

  • The mother of five needs to decide if kids need coats or not.
  • Teachers can plan outdoor activities based on weather information.
  • Ice Cream shop can forecast sales using the website weather information.
  • Citizens can report real-time storm information.
  • Using the website information families can plan weekends together.
